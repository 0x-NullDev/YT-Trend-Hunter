## 🧠 Core Insight

**Most YouTube tools analyze what already happened.**

TubeBuddy, VidIQ, and every other tool on the market looks at **past performance** — what videos already went viral, what keywords already rank, what channels already grew.

**YT Trend Hunter is different.**

We focus on:

👉 **What is about to happen** — trends forming right now
👉 **Where demand is forming** — before supply catches up
👉 **What creators are not seeing yet** — hidden opportunities in comments, searches, and gaps

We don't tell you what worked yesterday.

We tell you **what will work tomorrow.**

## 📦 Quick Start

### Prerequisites

- Docker & Docker Compose
- Python 3.11+ (for local development)
- Node.js 18+ (for frontend development)
- YouTube Data API v3 key

### 1. Get Your API Keys

Before starting, you need to obtain the following API keys:

#### 🔑 YouTube Data API v3 (Required)
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select existing
3. Enable **YouTube Data API v3** (Search: "YouTube Data API v3" → Enable)
4. Go to **Credentials** → **Create Credentials** → **API Key**
5. Copy the key — this is your `YOUTUBE_API_KEY`
6. *(Optional)* Restrict the key to "YouTube Data API v3" only for security

#### 🤖 AI Provider (Optional — for AI-powered analysis)
Choose one:

| Provider | How to Get Key | Cost |
|----------|---------------|------|
| **DeepSeek** | [platform.deepseek.com](https://platform.deepseek.com/) → API Keys | ~$0.14/1M tokens |
| **OpenAI** | [platform.openai.com](https://platform.openai.com/api-keys) | ~$0.50/1M tokens |
| **Anthropic** | [console.anthropic.com](https://console.anthropic.com/) | ~$3.00/1M tokens |
| **Ollama** | No key needed — runs locally | Free |

> **Note:** The system works **without an AI provider** using rule-based analysis. AI enhances the results.

#### 🔴 Reddit API (Optional)
1. Go to [reddit.com/prefs/apps](https://www.reddit.com/prefs/apps/)
2. Click **"Create App"** → select **"script"**
3. Copy the `client_id` (under the app name) and `client_secret`

### 2. Clone & Configure

```bash
git clone https://github.com/0x-NullDev/YT-Trend-Hunter.git
cd YT-Trend-Hunter

# Copy environment file
cp .env.example .env

# Edit .env with your API keys
# At minimum, set YOUTUBE_API_KEY
```

### 2. One-Click Install

Choose your platform:

```bash
./install.sh
```

```bash

docker-compose logs -f backend
```

- **Frontend**: http://localhost:3000

- **API Docs**: http://localhost:8000/docs
- **Redoc**: http://localhost:8000/redoc

## 📊 Scoring Formulas

### Trend Strength (0-100)
```
TS = w1 × GV_norm + w2 × EV_norm + w3 × SM_norm + w4 × VV_norm
```
- GV = Growth Velocity
- EV = Engagement Velocity
- SM = Search Momentum
- VV = Video Velocity

### Opportunity Score (0-100)
```
OS = TS × 0.30 + (100 - CL) × 0.25 + (100 - SS) × 0.25 + AD × 0.20
```
- TS = Trend Strength
- CL = Competition Level
- SS = Saturation Score
- AD = Audience Demand

### Content Gap Score (0-100)
```
GAP = demand_score × 0.50 + supply_gap × 0.30 + demand_momentum × 0.20
```

### Viral Score (0-100)
```
VS = view_velocity × 0.40 + engagement_rate × 0.30 + like_ratio × 0.20 + comment_ratio × 0.10
```
